# Claude Recall Persistent Conversation

This Python script demonstrates how to give Claude a persistent memory by saving and loading conversation history to a local JSON file. This allows for continuous project discussions without losing context between sessions.

## Language

`python`

## How to Run

1. Set your ANTHROPIC_API_KEY environment variable.
2. Run the script: python claude_recall.py
3. Interact with Claude; your conversation will be saved.
